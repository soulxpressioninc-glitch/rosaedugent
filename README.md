# RosaEduAgent 🏫

**AI-powered school quality assistant for Nigerian schools — built by Rosa Impact Solutions**

## What It Does

RosaEduAgent helps school leaders and quality coordinators improve teacher performance. A school leader types in their challenge and RosaEduAgent searches the school's teacher database, identifies who needs support, and generates a structured improvement plan.

## The Problem It Solves

In Nigeria, school quality collapses silently. Teachers struggle, leaders are overwhelmed, and no one connects the dots between what's happening in classrooms and what needs to happen next. RosaEduAgent closes that gap.

## How It Works

1. School leader asks a question in natural language
2. RosaEduAgent queries the MongoDB teacher database
3. Agent identifies struggling teachers by name
4. Agent generates a targeted improvement plan
5. Plan is assigned with follow-up recommendations

## Tech Stack

| Layer | Technology |
|---|---|
| AI Reasoning | Google Gemini |

| Database | MongoDB Atlas |
| Backend | Python / Flask |

## Example Queries

- "Which teachers need improvement?"
- "Which teachers are struggling with literacy?"
